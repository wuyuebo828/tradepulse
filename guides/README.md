# TradePulse Weekly Articles

Put weekly article HTML files in this folder:

```text
E:\Roblox\guides\
```

Recommended filename format:

```text
grow-a-garden-topic-name-2026-06-01.html
```

Each article should include:

- A unique `<title>` and meta description.
- One clear H1.
- Links back to `/values.html`, `/trade-calculator.html`, `/mutations.html`, or `/codes.html`.
- No Roblox passwords, account collection, or private user information requests.

After adding a new article:

1. Ask Codex to run the article update workflow, or run:

```text
node scripts/update-articles-index.js
```

2. Commit and push with GitHub Desktop.
