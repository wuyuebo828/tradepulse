# Fortnite Tools Backlog

## 定位

Fortnite 海外热度很高，但竞争也强。更适合做信息聚合和轻工具，不建议第一阶段就做完整大站。

## 玩家需求

高频需求：

- 今天商店有什么皮肤？
- 某个皮肤什么时候返场？
- 当前赛季任务怎么做？
- 活动什么时候开始或结束？
- 地图变化在哪里？
- XP 怎么刷更快？
- 某个武器当前强不强？

## 可做工具

第一批工具：

- Item Shop Tracker
- Skin Return Tracker
- Quest Checklist
- Event Countdown
- XP Quest Planner
- Map Change Notes

第二批工具：

- Locker Wishlist
- Cosmetic Collection Tracker
- Weapon Stats Comparison
- Creative Map Discovery

## 页面结构

- `/fortnite`
- `/fortnite/item-shop`
- `/fortnite/skins/:skin`
- `/fortnite/quests`
- `/fortnite/events`
- `/fortnite/map`
- `/fortnite/guides`

## 数据来源

- Fortnite 官方公告。
- Epic 公开信息和社媒。
- 可靠的第三方 API，需要确认使用条款。
- 社区爆料内容要谨慎标注，不要当成确定事实。

## 竞争和机会

竞争站很多，机会在于：

- 页面更快、更轻。
- 手机端体验更好。
- 任务清单做得更顺手。
- 把商店、任务、活动日历合在一起。

## 风险

- 皮肤、图片和商标版权敏感。
- 数据更新频率高。
- 需要避免使用未经授权的泄露内容作为确定信息。
- API 可用性和条款可能变化。
